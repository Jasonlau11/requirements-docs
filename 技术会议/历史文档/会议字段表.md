# 会议字段表

| 字段名称 | 填写方式 | 选项/规则 | 显示条件 | 备注 |
|---|---|---|---|---|
| 会议名称 | 手动填写 | 字数上限 50 | 默认显示 | - |
| 会议形式 | 单选 | 线上、线下、线上+线下 | 默认显示 | - |
| 会议场景 | 单选 | 开发者会议、产业会议、产品发布会议、区域营销会议、高校会议 | 默认显示 | - |
| 会议举办地域 | 单选 | 省市列表 | 默认显示 | - |
| 会议举办地点 | 手动填写 | 自由输入 | 默认显示 | - |
| 会议召开时间 | 手动选择 | 年月日时分 | 默认显示 | - |
| 会议时长 | 单选 | 半天、1天、2天、3天、3天以上 | 默认显示 | - |
| 会议规模 | 单选 | 50人以下、50-200人、200-500人、500人以上 | 默认显示 | - |
| 会议举办频率 | 单选 | 定期举办、单次举办 | 默认显示 | - |
| 所属产业 | 多选 | 可选范围支持修改 | 当 `会议场景=产业会议` 时显示 | 条件字段 |
| 涉及产品 | 手动输入 | 自由输入 | 当 `会议场景=产品发布会议` 时显示 | 条件字段 |
| 涉及区域 | 多选 | 北京、上海、深圳、广州、武汉、杭州、成都、西安、南京 | 当 `会议场景=区域营销会议` 时显示 | - |
| 涉及高校 | 多选 | 高校列表 | 当 `会议场景=高校会议` 时显示 | 条件字段 |
| 开发者类型 | 多选 | 前端开发、后端开发、全栈开发、移动端开发、运维/DevOps、大数据/AI、嵌入式/物联网、游戏开发、其他 | 默认显示 | - |
| 会议类型 | 单选 | 技术峰会、技术沙龙、技术研讨会 | 默认显示 | - |
| 目标人群 | 多选 | 开发者、架构师、产品经理、技术管理者、学生、泛技术人群 | 默认显示 | - |
| 常用标签 | 多选 | 可选范围为常用标签库，支持自定义标签 | 默认显示 | - |
| 会议简介 | 手动填写 | 支持富文本 | 默认显示 | - |
| 会议背景图 | 上传/选择 | 支持上传图片或选择某个图片 | 默认显示 | - |
| 会议结构类型 | 单选 | 大型会议、小型会议 | 默认显示 | - |
| 会议日程 | 手动填写 | 日期-环节-分会场-议题四层结构 | 默认显示 | 建议后续拆分为结构化子表 |
| 其他内容载体形式 | 手动填写 | 自由输入 | 默认显示 | - |
| 联系人 | 手动填写 | 自由输入 | 默认显示 | - |
| 联系电话 | 手动填写 | 自由输入 | 默认显示 | 建议后续增加手机号格式校验 |
| 公司 | 单选 | 可选范围为常见公司列表 | 默认显示 | - |
| 部门 | 手动填写 | 自由输入 | 默认显示 | - |
| 职位 | 手动填写 | 自由输入 | 默认显示 | - |
