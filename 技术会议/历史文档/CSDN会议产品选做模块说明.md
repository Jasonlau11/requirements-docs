## 二、选做 / 本期不包含模块

| 模块 | 说明 |
|------|------|
| **管理端数据看板** | 会议审核管理端「数据看板」页签：创建企业统计、地域统计、行业统计、参会用户统计等报表。本期仅做基础数据看板，其他统计维度与图表待后续定义与开发。 |
| **推荐干预（加权模式）** | 对会议在推荐位或排序上的运营干预能力。本期不开发。 |
| **账单管理与发票申请** | 办会方账单列表查询、发票申请等前端功能。本期不提供；费用记录可预留供后续对账使用。 |
| **推广长图生成** | 生成含会议核心信息与二维码的推广长图（V2 规划）。本期仅预留入口，不实现生成与下载。 |
| **移动端管理员审核** | 会议审核、驳回、下架等管理能力仅在网页端提供；移动端不包含管理端功能。 |
| **线下商务流程深度打通** | 与线下签单、合同、发票等传统商务流程的深度打通。本期无明确要求则视为不包含。 |
